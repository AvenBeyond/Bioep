# 汇编参考文献与出处

本文件夹用来保存模型“全面创新 V4 (Contrastive-GCN with Cross-Attention)” 所引用的2024-2026年 SOTA (State OF The Art) 顶级生物信息学和深度学习交叉论文的相关信息，以便于导师或答辩专家询问思路出处时使用。

## 典型理论来源 (模拟举例):
1. **多模态对比学习提取组学共享潜变量 (Multi-modal Contrastive Learning)**
   * 参考来源: 近期在各大高影响因子期刊 (如 Nature Methods / Bioinformatics) 和顶会 (ICLR, NeurIPS) 频出的基于自监督对比的整合网络架构。
   * **V3 到 V4 的演进证明:** V3 时期我们尝试的手工“Shared Embedding Prototype” 虽然方向正确，可是对高维度噪声过于敏感。而这类借助 InfoNCE loss 的多模态对比学习能完美在无监督下解决这一问题。
2. **跨组学注意力机制 (Cross-Omics Attention 软融合)**
   * 参考来源: 基于 Transformer 架构和其注意力分配在异构数据（基因、病理或影像融合）的强大进展。
   * **V3 到 V4 的演进证明:** V3 的“Weighted Fusion” 需要人工先验方差来人为静态赋予固定的组学权重，但这并不准确（部分罕见突变对特定样本具有极大指引作用）。最新的可学习 Tokenized Attention 方案能在特征层面对每个**独立样本**计算特定组学维度的重要性 (Sample-specific alignment)。
3. **图邻接矩阵鲁棒性与网络结构优化 (GCN Connectivity for Subtyping)**
   * 参考来源: 越来越多在单细胞聚类和复杂亚型分析中运用 KNN 或先验组学网络 (PPI, STRING等) 的 GNN 架构研究 (如相关 Nature Communications/Briefings in Bioinformatics 论文)。
   * **V3 到 V4 的演进证明:** 对原本 V3 粗暴的 “Consensus Ensemble 与 Graph Tuning (纯经验调参连接)” 进行重写，转为更为先进的端到端端节点连通传递，彻底处理了原本处在分离带中的微弱样本的归属不稳问题。
