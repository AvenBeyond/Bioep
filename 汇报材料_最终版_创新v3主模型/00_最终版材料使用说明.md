# 最终版材料使用说明

本文件夹是本次汇报的最终整理版，最终主模型已确定为创新模型：

`innovation_v3_clinical_consensus`

传统模型 `early_fusion_kmeans@main5,k=2` 不再作为最终主模型，而作为保守传统基线用于对照。

## 文件结构

- `01_实验汇报详细文档.md`：按实验汇报要求整理的完整中文报告，可作为讲稿和报告主体。
- `02_PPT制作建议.md`：5-10 分钟汇报的 PPT 页序、配图和讲解重点。
- `03_图片与表格索引.md`：说明每张图、每个表的用途。
- `图片/`：已复制的最终版汇报图片，可直接插入 PPT。
- `关键结果表/`：已复制的最终版核心 CSV 结果表。
- `参考文献与出处/`：文献和数据来源链接。

## 最终汇报口径

1. 我们先建立传统多组学基线模型，确认 TCGA-STAD 五组学数据可以得到有临床关联的二分型结构。
2. 在传统模型基础上，进一步构建创新 v3：共享潜变量表示（shared latent representation）+ 边界样本共识细化（boundary consensus refinement）。
3. 经过随机种子敏感性复核，创新 v3 标签高度稳定，并在 7 个核心汇报指标上超过传统基线，因此最终主模型确定为创新 v3。
4. 需要明确边界：v3 是内部探索性临床感知分型，仍需外部独立队列验证后才能讨论临床部署。
